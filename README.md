**Group Members:** [Joshua Samad Trujillo Lagos, Sebastián Ramos, Santiago Sandoval]

## Overview
This repository contains the solution for the Academic (AD) database workshop. Three tasks are addressed:

1. **Student Grade Drop Analysis** – Simple SQL queries and analysis to identify factors behind perceived grade drops.
2. **Faculty Administrative Capacity Improvement** – Data extraction and proposals to enhance faculty administration.
3. **Preparation for New Academic Season** – Comprehensive data overview, missing info detection, and recommendations.

All SQL is written for Oracle and uses the exact table/column names from the AD schema.

## How to Use
- Run the SQL scripts in Oracle Live SQL in the order provided.
- Export query results to CSV and create graphs using Excel, Python, or any tool.
- Refer to the analysis PDFs for interpretations and proposals.
